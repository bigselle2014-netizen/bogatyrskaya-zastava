# VFXSpec — Визуальные эффекты
**Версия**: 1.0 | **Дата**: Апрель 2026 | **Фаза**: 2 (Месяцы 3–5)

Спецификации 7 типов VFX-спрайтов для боевой системы, смерти врагов и ресурсов.

---

## Общие требования

| Параметр | Значение |
|---------|---------|
| Формат | PNG-24 с Alpha-каналом |
| Компрессия | ASTC 6x6 |
| Blend Mode | Additive (по умолчанию, если не указано иное) |
| Передача | Горизонтальные spritesheets (кадры в ряд) |
| Именование | `vfx_{type}_sheet.png` |
| Упаковка | SA_VFX (1024x512) |

## VFX-01: Удар мечом (Дружина)

**ID**: vfx_sword_slash
**Фракция**: Дружина (DR)
**Тип**: Slash arc — дуговой след от клинка

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 128x128px |
| Кадров | 3 |
| Spritesheet | 384x128px |
| Длительность | 0.25 сек (≈0.083 сек/кадр) |
| Blend Mode | Additive |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Основной (сталь) | `#2E5FA3` | Центр дуги |
| Блик | `#7FB8FF` | Светлый край дуги |
| Белый | `#FFFFFF` | Самый яркий блик |

### Покадровое описание

**Кадр 1 — Начало дуги**: тонкая дуга `#FFFFFF` (1px) длиной 48px, начинается от нижнего правого угла, идёт вверх-влево. Слабое свечение `#7FB8FF` opacity 40% radius 8px вдоль дуги.

**Кадр 2 — Пик**: дуга расширяется до 3 линий (3px, 2px, 1px), длина 80px. Основная — `#FFFFFF`, средняя — `#7FB8FF`, тонкая — `#2E5FA3`. Свечение `#7FB8FF` radius 16px. Максимальная яркость.

**Кадр 3 — Затухание**: дуга рассыпается на 4-5 осколков `#7FB8FF` (4px каждый, opacity 50%). Свечение уменьшается (radius 8px, opacity 20%). Исчезновение.

## VFX-02: Огненный взрыв (Берсерки)

**ID**: vfx_fire_burst
**Фракция**: Берсерки (BE)
**Тип**: Radial burst — радиальный огненный взрыв

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 256x256px |
| Кадров | 5 |
| Spritesheet | 1280x256px |
| Длительность | 0.5 сек (0.1 сек/кадр) |
| Blend Mode | Additive |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Ядро | `#FFFFFF` | Центр взрыва |
| Горячее пламя | `#FFD700` | Внутренний слой |
| Основное пламя | `#FF6D00` | Средний слой |
| Внешнее пламя | `#FF4400` | Внешний край |

### Покадровое описание

**Кадр 1 — Зарождение**: точка `#FFFFFF` (8px diameter) в центре. Вокруг — 4 искры `#FFD700` (3px) разлетаются по диагоналям (16px от центра).

**Кадр 2 — Расширение**: сфера `#FFD700` (32px diameter). Языки пламени `#FF6D00` (8 штук, 16px) расходятся радиально. Искры дальше (32px).

**Кадр 3 — Пик**: максимальный размер — radius 64px. Центр `#FFFFFF` → `#FFD700` → `#FF6D00` → `#FF4400` (кольцевой градиент). 12 языков пламени `#FF4400` по краям (24px каждый). Дым `#333333` opacity 20% за пламенем.

**Кадр 4 — Рассеивание**: сфера сжимается (radius 48px). Языки пламени отделяются — 6 огненных частиц `#FF6D00` (8px) разлетаются за пределы сферы. Дым усиливается (opacity 30%).

**Кадр 5 — Затухание**: только дым `#333333` opacity 20% (radius 32px) и 3-4 затухающие искры `#FF4400` opacity 30%. Исчезновение.

## VFX-03: Стрела полёт (Лучники)

**ID**: vfx_arrow_trail
**Фракция**: Лучники (LU)
**Тип**: Arrow trail — след летящей стрелы

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 128x64px |
| Кадров | 4 |
| Spritesheet | 512x64px |
| Длительность | 0.3 сек (0.075 сек/кадр) |
| Blend Mode | Additive |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Древко | `#795548` | Тело стрелы |
| Наконечник | `#C0C8D4` | Металл |
| Яд/оперение | `#2E7D32` | Зелёное свечение |
| След | `#43A047` | Trail за стрелой |

### Покадровое описание

**Кадр 1 — Выпуск**: стрела целиком (древко `#795548` 48x3px, наконечник `#C0C8D4` 8x4px, оперение `#2E7D32` 12x6px). Вспышка у оперения `#43A047` radius 8px (энергия выстрела).

**Кадр 2 — Полёт**: стрела смещена влево на 32px. За ней — зелёный trail `#43A047` opacity 40% (3 линии, 32px длиной). Наконечник чуть светится `#C0C8D4` opacity 80%.

**Кадр 3 — Ускорение**: стрела ещё левее (скорость). Trail удлиняется (48px), яд `#2E7D32` — капли (`#43A047`, 2 штуки, 3px) отлетают от стрелы.

**Кадр 4 — Попадание**: стрела воткнулась (только оперение торчит). Вспышка при попадании `#43A047` radius 16px. Ядовитое облачко `#2E7D32` opacity 30% (16x16px) — начало DoT.

## VFX-04: Магический эффект (Волхвы)

**ID**: vfx_arcane_circle
**Фракция**: Волхвы (VL)
**Тип**: Arcane circle — магический круг заклинания

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 256x256px |
| Кадров | 6 |
| Spritesheet | 1536x256px |
| Длительность | 0.6 сек (0.1 сек/кадр) |
| Blend Mode | Additive |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Основной (магия) | `#6B2FA0` | Линии круга |
| Свечение | `#9B5FD0` | Заполнение, блики |
| Руны | `#F0C040` | Рунические символы |
| Белый | `#FFFFFF` | Центральная вспышка |

### Покадровое описание

**Кадр 1 — Появление**: тонкий круг `#6B2FA0` (1px) radius 48px в центре кадра. 4 руны `#F0C040` (10px каждая) появляются по кардинальным точкам круга (N/S/E/W). Opacity 40%.

**Кадр 2 — Формирование**: круг ярче (2px, opacity 80%). Внутри — второй круг radius 32px. Руны вращаются по часовой (смещены на 22.5 градуса). Свечение `#9B5FD0` opacity 20% заполняет внутренний круг.

**Кадр 3 — Активация**: оба круга полностью яркие. 8 рун (добавились 4 новые). Линии связи между рунами `#6B2FA0` (1px). Центр — точка `#FFFFFF` (6px). Магический гул.

**Кадр 4 — Пик**: центральная вспышка `#FFFFFF` radius 24px. Круги расширяются (radius 64px и 48px). Руны максимально яркие `#F0C040` opacity 100%. Из центра — лучи `#9B5FD0` (8 лучей, 48px, 1px).

**Кадр 5 — Разряд**: лучи удлиняются до 96px. Руны начинают рассыпаться (2 руны стали частицами). Круги истончаются (1px). Центральное свечение ослабевает.

**Кадр 6 — Затухание**: круги распадаются на дуги (4 дуги, opacity 30%). Руны — разлетевшиеся частицы `#F0C040` (6 штук, 3px). Центр — `#9B5FD0` opacity 10%. Полное исчезновение.

## VFX-05: Исцеление (Знахари)

**ID**: vfx_heal_particles
**Фракция**: Знахари (ZN)
**Тип**: Green particles rise — зелёные частицы поднимаются вверх

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 128x128px |
| Кадров | 5 |
| Spritesheet | 640x128px |
| Длительность | 0.5 сек (0.1 сек/кадр) |
| Blend Mode | Additive |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Целебный зелёный | `#43A047` | Основные частицы |
| Светлый зелёный | `#80FF80` | Яркие частицы |
| Белый | `#FFFFFF` | Блики исцеления |
| Природный | `#2E7D32` | Тёмные частицы |

### Покадровое описание

**Кадр 1 — Зарождение**: у нижнего края кадра — 6 мелких частиц `#43A047` (4px каждая, кружки). Расположены веером (ширина 64px). Слабое свечение `#80FF80` opacity 20% radius 8px вокруг каждой.

**Кадр 2 — Подъём**: частицы поднялись на 24px. Появились ещё 4 частицы `#80FF80` (3px, ярче) между основными. Свечение усиливается (opacity 40%). Лёгкое покачивание — частицы не летят прямо, а волнообразно.

**Кадр 3 — Расцвет**: пик — все частицы на высоте 48-64px. 2 самые яркие `#FFFFFF` (5px). Зелёные полосы `#43A047` (2px, 16px длиной) — хвосты за частицами. Общее свечение radius 48px opacity 30%.

**Кадр 4 — Растворение**: частицы на 72-88px. Начинают тускнеть (opacity 60%). Хвосты удлиняются. Добавляются мелкие листочки `#2E7D32` (3 штуки, 6x4px) — кружатся.

**Кадр 5 — Исчезновение**: частицы у верхнего края, opacity 20%. Листочки ещё видны (opacity 40%). Последние блики `#FFFFFF` (2 штуки, 2px). Полное затухание.

## VFX-06: Смерть врага

**ID**: vfx_enemy_death
**Тип**: Dissolve — растворение/рассеивание
**Универсальный** (не привязан к фракции)

### Параметры

| Параметр | Значение |
|---------|---------|
| Размер кадра | 128x128px |
| Кадров | 4 |
| Spritesheet | 512x128px |
| Длительность | 0.4 сек (0.1 сек/кадр) |
| Blend Mode | Normal (не Additive — враги темнее) |

### Палитра

| Цвет | HEX | Применение |
|------|-----|-----------|
| Тёмная пыль | `#333333` | Основные частицы распада |
| Фиолетовый (нечисть) | `#3D1A5C` | Магическая пыль |
| Свечение глаз | `#FF4400` | Последнее свечение глаз |
| Белый | `#FFFFFF` | Финальная вспышка |

### Покадровое описание

**Кадр 1 — Удар**: враг «замирает» (общий силуэт). Глаза `#FF4400` вспыхивают ярче (radius 6px). По телу — трещины `#3D1A5C` (4-5 линий, 1px).

**Кадр 2 — Распад начало**: нижняя четверть тела рассыпается в частицы `#333333` (8-10 частиц, 3-6px). Фиолетовая пыль `#3D1A5C` opacity 40% поднимается. Глаза ещё горят.

**Кадр 3 — Распад пик**: тело на 70% рассыпалось. Частицы `#333333` (16-20 штук) разлетаются во все стороны. Глаза — последнее что исчезает (opacity 50%). Фиолетовый дым `#3D1A5C` radius 32px.

**Кадр 4 — Исчезновение**: последние частицы (4-5 штук, opacity 20%) растворяются. Маленькая вспышка `#FFFFFF` (4px) — финальный штрих. Ничего не остаётся (чисто — рейтинг 6+).

| Цвет | HEX | Применение |
|------|-----|-----------|
| Золото | `#FFD700` | Монета/руна |

| Белый | `#FFFFFF` | Искры |

**Кадр 3 — Подбор**: монета «улетает» к HUD (сжимается до 8px, смещается вверх-вправо). Золотой trail `#FFD700` opacity 50% (24px дуга). Финальная вспышка `#FFF0A0` (6px). Тень исчезает.

## Хендовер → Unity Dev Middle

**Что передаётся**: 7 spritesheet-файлов.

**Список файлов**:
```
vfx_sword_slash_sheet.png    (384x128px)
vfx_fire_burst_sheet.png     (1280x256px)
vfx_arrow_trail_sheet.png    (512x64px)
vfx_arcane_circle_sheet.png  (1536x256px)
vfx_heal_particles_sheet.png (640x128px)
vfx_enemy_death_sheet.png    (512x128px)
vfx_resource_pickup_sheet.png (192x64px)
```

**Формат передачи**: ZIP `vfx-v{N}.zip`, папка `vfx/`.

**Что Dev делает**:
1. Импортирует в `Assets/Art/Sprites/VFX/`
2. Sprite Mode: Multiple, Slice по Grid (размеры указаны для каждого)
3. Упаковывает в `SA_VFX` (1024x512)
4. Настраивает Animator Controller для каждого VFX
5. Blend Mode:
   - Additive: vfx_sword_slash, vfx_fire_burst, vfx_arrow_trail, vfx_arcane_circle, vfx_heal_particles, vfx_resource_pickup
   - Normal: vfx_enemy_death
6. Sorting Layer: «VFX» (над персонажами, под UI)
7. ASTC 6x6 компрессия
8. Проверить на Redmi A2+: VFX не вызывает просадки FPS

**Критерии приёмки**:
- [ ] Все VFX читаемы на фоне каждого из 8 биомов
- [ ] Additive blend не создаёт «белых пятен» при наложении нескольких VFX
- [ ] Анимации плавные (нет рывков между кадрами)
- [ ] vfx_enemy_death соответствует рейтингу 6+/12+ (нет крови, только пыль)
- [ ] Цвета VFX соответствуют палитрам фракций из StyleGuide.md
- [ ] vfx_resource_pickup: золотая монета читается на тёмных и светлых биомах
