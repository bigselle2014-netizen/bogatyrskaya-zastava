# UISpec — Полный UI для MVP
**Версия**: 1.0 | **Дата**: Апрель 2026 | **Фаза**: 2 (Месяцы 3–5)

Полная спецификация пользовательского интерфейса для MVP «Богатырская Застава». Все экраны, все элементы.

---

## Общие правила UI

| Параметр | Значение |
|---------|---------|
| Целевое разрешение | 1080x1920px (portrait) |
| Масштабирование | Canvas Scaler: Scale With Screen Size, Reference 1080x1920, Match Width Or Height 0.5 |
| Safe area (notch) | Верхний отступ: 88px (iPhone notch), нижний: 48px (gesture bar) |
| Стиль | Cel-shading, деревянные рамки с русской резьбой |
| Обводка всех элементов | 2px `#1A1A1A` |
| Шрифт основной | Без засечек с кириллицей (Russo One / Exo 2 / Rubik) |
| Шрифт мин. размер | 14px при 1080p, 11px при 720p |
| Кнопки | Скруглённые углы 8px, текстура тёмного дерева |
| Фон UI-панелей | `#1A1A2E` opacity 90% |
| Рамки | 4-6px, `#C8860A` (золото) |

**Всегда на экране** во время боя

**Позиция**: нижний край экрана, safe area bottom 48px
**Фон**: панель `#1A1A2E` opacity 90%, высота 128px

| Элемент | Описание |
|---------|----------|

### Дорожка 30 уровней

**Горизонтальный скролл**

**Структура одного узла** (144px шириной):

```
┌─────────────────┐
│ [ПЛАТНЫЙ ПРИЗ]  │  ← 64x64px, рамка #FFD700
│                 │
│  ── Ур.{N} ──  │  ← номер уровня, 14px
│                 │
│ [БЕСПЛАТ.ПРИЗ]  │  ← 64x64px, рамка #4A4A5A
└─────────────────┘
```

**Визуал дорожки**: горизонтальная линия `#C8860A` (2px) соединяет узлы. Заполненные — золотые `#FFD700`, незаполненные — серые `#4A4A5A`.

**Что передаётся**: ~90 PNG файлов + Figma-исходники.


**Что Dev делает**:
1. Импортирует в `Assets/Art/Sprites/UI/`
2. Упаковывает в `SA_UI_Full` (2048x2048)
3. Canvas Scaler: Scale With Screen Size, 1080x1920, Match 0.5
4. Safe Area: учитывать отступы 88px top, 48px bottom
5. ASTC 6x6 компрессия
6. Проверить на Redmi A2+ (720p) — все тексты читаемы, кнопки нажимаемы (min tap target 48px)

**Критерии приёмки**:
- [ ] Все тексты >= 14px при 1080p (11px при 720p)
- [ ] Кнопки >= 48x48px (минимальная зона нажатия)
- [ ] Safe area: контент не перекрывается notch/gesture bar
- [ ] Цветовая схема консистентна со StyleGuide.md
- [ ] HUD не перекрывает более 15% игрового поля
- [ ] Магазин: все цены в рублях, формат понятен
- [ ] DeckBuilder: карты читаемы без увеличения
- [ ] Общий стиль: русская эстетика, деревянные рамки, золотые акценты
