# TilesetSpec — Все биомы
**Версия**: 1.1 | **Дата**: Апрель 2026 | **Фаза**: 1–2

Спецификации тайловых сетов для всех биомов.

---

## Общие технические параметры

| Параметр | Значение |
|---------|---------|
| Размер тайла | 128x128px |
| Формат | PNG-24 с Alpha-каналом |
| Компрессия | ASTC 6x6 |
| Совместимость | Unity 2D Tilemap, Rule Tile, Sprite Atlas |
| Тайлов на биом | 16 (4 ground + 4 path + 8 decor) |
| Parallax-слоёв на биом | 2 (1920x1080px каждый) |
| Именование тайлов | `tile_{biome}_{category}_{variant}.png` |
| Именование фонов | `bg_{biome}_layer{N}.png` |

### Структура тайлов для каждого биома

| Категория | Кол-во | Роль |
|-----------|--------|------|
| ground_01..04 | 4 | Основные тайлы земли/поверхности |
| path_straight, path_curve_l, path_curve_r, path_cross | 4 | Путь врагов |
| deco_01..08 | 8 | Декоративные объекты поверх ground/path |


**Что Dev делает**:
1. Тайлы → `Assets/Art/Tiles/{BiomeName}/`
2. Фоны → `Assets/Art/Backgrounds/`
3. Создать Unity Tile Palette для каждого биома
4. Упаковать в Sprite Atlas:
   - `SA_Tilesets_Act1` (Сёла у реки — 1 биом)
   - `SA_Tilesets_Act2` (3 биома: Дикое поле, Кикиморины топи, Пещеры Кощея)
   - `SA_Tilesets_Act3` (3 биома + Ирий: Полночный предел, Чёрная крепость, Ирий-сад)
5. ASTC 6x6 компрессия
6. Parallax: Layer1 speed 0.3, Layer2 speed 0.6
7. Проверить стыковку тайлов в каждом биоме
8. Проверить на Redmi A2+

**Критерии приёмки**:
- [ ] Ground-тайлы стыкуются без швов внутри каждого биома
