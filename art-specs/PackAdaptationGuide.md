# PackAdaptationGuide — Адаптация купленных паков
**Версия**: 1.0 | **Дата**: Апрель 2026 | **Фаза**: 1 (Недели 1–8)

Инструкция по адаптации купленных арт-паков под визуальный стиль «Богатырской Заставы». Два пака: враги ($20) и UI ($10).

---

## Общие принципы адаптации

**Цель адаптации**: сделать так, чтобы купленные ассеты не «выбивались» из стиля — играли в одной команде с оригинальным артом башен.

**Три кита адаптации**:
1. **Цвет** — перекрасить в палитру проекта (холодные тона для врагов, золото+дерево для UI)
2. **Обводка** — заменить родную обводку на проектную (2px `#1A0A2E` для врагов, 3px `#8B6914` для UI)
3. **Детали** — убрать лишнее, добавить специфическое (глаза нечисти, русские орнаменты)

**Инструменты** (рекомендуемые):
- Adobe Photoshop CC 2024+: основной инструмент
- Aseprite: если пак в пиксель-арте (проверить при покупке)
- GIMP (бесплатная альтернатива): поддерживает все нужные операции

**Бюджет**: $20  
**Рекомендуемые паки** (один из):

### Что использовать, что нет

**Подходят**:
- Скелеты-гуманоиды (прямоходящие)
- Призраки/духи (расплывчатые, полупрозрачные)
- Летучие существа (птицы, летучие мыши, вороны)
- Зомби/нежить в лохмотьях (без современной одежды)
- Гоблины/импы (мелкие, уши, когти)

**Не подходят — исключить**:
- Монстры в современной одежде (рубашки, джинсы, галстуки)
- Роботы, киборги, sci-fi существа
- Аниме-персонажи с нереалистичными пропорциями (если стиль сильно отличается)
- Существа в ярко-розовом, ярко-жёлтом, неоновых цветах (без возможности перекраски)
- Слишком детализованные (фотореалистичные) — выбьются из cel-shading стиля

### Алгоритм адаптации врага (Photoshop)

**Время на 1 врага (idle + run = 2 кадра)**: 1.5–2.5 часа

#### Шаг 1 — Подготовка холста

1. Открыть исходный спрайт врага из пака
2. Проверить текущий размер. Если исходник больше 256×256 — уменьшить: `Image → Image Size` → **128×128px** (для пехоты) или **96px** (для быстрого, но всё равно холст 128×128)
3. Если исходник меньше 128×128 — увеличить. Если исходник пиксель-арт — использовать `Nearest Neighbor` при увеличении (не бикубическое!)
4. Убедиться, что спрайт на прозрачном фоне (Layer → Transparency). Если фон цветной — удалить через `Select → Color Range` или `Magic Wand`

#### Шаг 2 — Цветокоррекция (основная)

**Для E_PESHIY / E_BRONEY** (болотно-синюшный тон):
```
1. Layer → New Adjustment Layer → Hue/Saturation
   Hue: +140 (если исходник коричневый/красный → синий)
   Saturation: –20 (приглушить)
   Lightness: –10 (темнее)

2. Layer → New Adjustment Layer → Color Balance
   Shadows: Cyan +10, Magenta 0, Blue +15
   Midtones: Cyan +5, Blue +8
   Highlights: без изменений

3. New Layer → Mode: Multiply → Opacity: 35%
   Залить цветом #3D1A5C (тёмно-фиолетовый)
   Это "обволакивает" персонажа фиолетовой тьмой
```

**Для E_BYSTR** (зелёный кикиморёнок):
```
1. Hue/Saturation:
   Hue: +100 (сдвиг к зелёному от любого исходного тона)
   Saturation: +20 (насыщенный зелёный)
   Lightness: 0

2. Color Balance → Shadows: Green +10, Blue –5
3. Multiply-слой #1A3B1A (болотный) opacity 25%
```

**Для E_LETUN** (вороно-чёрный с синим переливом):
```
1. Desaturate: Image → Adjustments → Desaturate (полностью обесцветить)
2. Hue/Saturation: Colorize ON, Hue: 225, Saturation: 20, Lightness: –40
   (даёт тёмно-синий тон)
3. Curves: Input 200 → Output 140 (затемнить средние тона)
4. Multiply-слой #1A1A2E opacity 50%
```

#### Шаг 3 — Добавление деталей (специфика проекта)

**Глаза нечисти** (ОБЯЗАТЕЛЬНО для всех врагов):
1. Найти позицию глаз на спрайте
2. Создать новый слой поверх всех
3. Инструмент Brush: 4–6px (для 128px спрайта), жёсткость 80%
4. Цвет: `#FF4400` (оранжево-красный)
5. Нарисовать 2 точки-глаза
6. Добавить Outer Glow к слою: цвет `#FF6600`, size 4–6px, opacity 70%

**Для E_PESHIY — соломенная шляпа**:
1. Нарисовать поверх головы простую форму: трапеция (поля) + конус (тулья)
2. Цвет: `#D4A020` (солома), тень `#9A6A10`
3. 2–3 торчащих соломины (тонкие линии), длина 8–12px
4. Дыры в шляпе: 2–3 дыры (удалить ластиком небольшие участки)

**Для E_BRONEY — мох на доспехах**:
1. Новый слой с режимом Multiply
2. Цвет кисти `#1A3B1A` (тёмно-зелёный), opacity 60%, жёсткость 0 (мягкая кисть)
3. Размазать 3–5 пятен мха на плечах и груди, диаметр каждого 8–16px

**Для E_BYSTR — хвост**:
1. Нарисовать мышиный хвост: изогнутая линия толщиной 3–4px у основания, сужается к концу до 1px
2. Длина: ~32–40px (выходит за правый край тела)
3. Цвет: `#3A5A2A` (как кожа), тень `#1A3A1A` по нижней стороне

**Для E_BYSTR — уши** (если в паке не было заострённых):
1. Liquify → Forward Warp: вытянуть существующие уши вверх и заострить
2. Или нарисовать поверх: острые треугольники 12–16px высотой, заострённые

**Для E_LETUN — глаза (человекоподобные)**:
1. Убрать/закрасить оригинальный птичий глаз
2. Нарисовать миндалевидный глаз 6×4px, цвет `#FF4400`
3. Узкий зрачок-щель `#1A0A2E` по центру

#### Шаг 4 — Обводка

**Убрать старую обводку** (если мешает):
1. Select → Color Range → выбрать цвет оригинальной обводки → Delete

**Добавить новую обводку 2px**:
1. Ctrl+Click по слою персонажа (выделить силуэт)
2. Select → Modify → Expand → 2px
3. Создать новый слой ПОД слоем персонажа
4. Fill: цвет `#1A0A2E`
5. Снять выделение

#### Шаг 5 — Финализация

1. Flatten All Layers
2. Проверить: спрайт на прозрачном фоне
3. Проверить размер холста: 128×128px
4. Сравнить с StyleGuide: глаза `#FF4400` видны, силуэт тёмный
5. Save As → PNG-24, максимальное сжатие
6. Назвать: `E_PESHIY_idle.png` или `E_PESHIY_run.png`

### Что использовать, что нет

**Используем**:
- Кнопки (любой формы — будем менять фон)
- Прогресс-бары (горизонтальные, с фоном и заливкой)
- Звёзды для оценки уровня (3 звезды)
- Рамки/панели (любые прямоугольные)
- Иконки (монеты, сердца, кристаллы)

**Не используем напрямую** (требуют адаптации или замены):
- Современные flat-иконки в стиле Material Design → перекрасить или заменить
- Кнопки в неоновых/bright-цветах → перекрасить
- Всё с sci-fi или аниме-элементами → убрать детали

---

### Алгоритм адаптации кнопок

**Время на 1 кнопку (normal + pressed)**: 30–60 минут

#### Шаг 1 — Подготовка

1. Открыть исходную кнопку из пака
2. Изменить размер до **256×64px** (или 128×64px для квадратных)
3. Разделить на слои (если пак предоставляет PSD) или работать с PNG

#### Шаг 2 — Замена фона кнопки на дерево

1. Создать новый слой «Дерево-фон»
2. Залить цветом `#3A2510` (тёмное дерево)
3. Добавить текстуру:
   - Filter → Noise → Add Noise: Amount 5%, Gaussian, Monochromatic
   - Filter → Blur → Motion Blur: Angle 0°, Distance 4px (имитация волокон дерева)
4. Режим слоя: Multiply 80% поверх оригинала кнопки (чтобы сохранить форму)

#### Шаг 3 — Рамка кнопки

1. Создать Border-слой
2. Select → Modify → Contract 0px (выделить весь холст)
3. Stroke: Width 3px, Color `#8B6914` (тёмное золото), Inside
4. Дополнительно — внешний Stroke 1px `#C8860A` (золото)

#### Шаг 4 — Металлические уголки

1. Нарисовать в каждом углу L-образную форму 10×10px
2. Цвет: `#C8860A` (золото), тень `#8B6914`
3. Центр уголка: круглая «заклёпка» диаметром 4px

#### Шаг 5 — Pressed-состояние

1. Дублировать слой Normal-кнопки
2. Image → Adjustments → Brightness/Contrast: Brightness –25
3. Перенести всё содержимое на 2px вниз (нажатие = кнопка «погружается»)

#### Шаг 6 — Иконка на кнопке

- Play ▶: белый треугольник `#F5F0E8` по центру, size 20×24px
- Pause ⏸: два белых прямоугольника `#F5F0E8`, 8×24px, промежуток 8px
- Settings ⚙: нарисовать шестерёнку `#F5F0E8`, или импортировать иконку и перекрасить
- Close ✕: два пересекающихся прямоугольника `#F5F0E8`, угол 45°
- Home 🏠: силуэт дома `#F5F0E8`, стиль лубочный (простой, крыша+стены)

### Алгоритм адаптации прогресс-бара

**Время на 1 прогресс-бар**: 20–30 минут

1. Открыть исходный прогресс-бар (фон + заливка — 2 файла)
2. Изменить размер: **512×24px** (или 256×16px для маленьких)
3. **Фон бара**: тёмное дерево `#2A1A0A`, рамка `#8B6914` 2px
4. **Заливка HP**: цвет `#C62828` (красный), блик `#E84040` (2px полоса сверху)
5. **Заливка Mana**: цвет `#6B2FA0` (фиолетовый), блик `#8B4FC0`
6. **Заливка XP**: цвет `#C8860A` (золото), блик `#F0A030`
7. Сохранить фон и заливку отдельно: `ui_progressbar_hp_bg.png` + `ui_progressbar_hp_fill.png`

---

### Алгоритм адаптации звёзд

**Время**: 30 минут на весь набор

1. Форма звезды из пака (или нарисовать с нуля — 5-лучевая)
2. **Заполненная звезда**: цвет `#FFD700` (золото), обводка `#7A5500` (тёмное золото) 2px
   - Внутренний блик: маленький треугольник `#FFF080` в верхнем левом углу звезды, opacity 70%
3. **Пустая звезда**: цвет `#3A3A3A` (тёмно-серый), обводка `#6A6A6A` 2px
4. Размер: 64×64px
5. Сохранить: `ui_icon_star_filled.png`, `ui_icon_star_empty.png`

---

### Алгоритм адаптации иконок ресурсов

**Время на 1 иконку**: 15–20 минут

**Монета (ui_icon_coin.png, 32×32px)**:
1. Круг диаметром 24px, цвет `#FFD700`
2. Обводка 2px `#7A5500`
3. Гравировка: тонкий силуэт воина или рубль `#F0C040`, 12×12px по центру

**Сердце (ui_icon_heart.png, 32×32px)**:
1. Классическая форма сердца `#C62828`
2. Блик: маленький овал `#FF6060` в верхнем левом, opacity 60%
3. Обводка 2px `#7B1111`

**Кристалл (ui_icon_crystal.png, 32×32px)**:
1. Форма: вытянутый шестиугольник (кристалл)
2. Цвет `#6B2FA0` (фиолетовый)
3. Блик: треугольник `#B060E0` справа-сверху

---

### Иконки башен (создаются вручную, не из пака)

**Время на 1 иконку**: 30–45 минут  
**Размер**: 64×64px

Иконки башен (ui_icon_{TowerID}.png) создаются с нуля — это упрощённые силуэты башен на цветном фоне.

**Шаблон для всех фракций**:
1. Круг диаметром 56px, центр холста
2. Залить цветом фракции (основной HEX):
   - DR: `#2E5FA3`, VL: `#6B2FA0`, LU: `#2E7D32`, BE: `#C62828`, ZN: `#F5F5F5`
3. Обводка круга 2px `#C8860A` (золото)
4. Поверх: белый силуэт башни ур.1 (упрощённый, только голова + основная деталь)
   - DR: шлем + меч/щит, VL: остроконечный колпак + посох, LU: шапка + лук
   - BE: голова + топор, ZN: платок + цветы на посохе
5. Силуэт: `#FFFFFF` opacity 90%, заливка без обводки
