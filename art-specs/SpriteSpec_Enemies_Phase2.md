# SpriteSpec — Враги Фазы 2
**Версия**: 1.0 | **Дата**: Апрель 2026 | **Фаза**: 2 (Месяцы 3–5)

Спецификации для 8 новых типов врагов. 3 модификатора (ядовитый, целитель, призрак) + 5 уникальных врагов для Актов 2-3 (рисуются с нуля).

---

## Общие требования

| Параметр | Значение |
|---------|---------|
| Размер холста | 128x128px (стандартные враги) |
| Формат | PNG-24 с Alpha-каналом |
| Компрессия | ASTC 6x6 |
| Анимация run | 6 кадров (spritesheet 768x128px) |
| Анимация death | 4 кадра (spritesheet 512x128px) |
| Глаза нечисти | `#FF4400` (обязательно для всех) |
| Обводка | 2px, цвет `#1A0A2E` |
| Движение | Влево по экрану |
| Именование | `E_{ID}_{state}_sheet.png` |

---

## МОДИФИКАТОРЫ (3 типа)

Модификаторы — вариации базовых врагов с добавленными визуальными эффектами. Могут применяться к любому базовому врагу.

## E_POISON — Ядовитый

**ID**: E_POISON
**Тип**: Модификатор — DoT аура
**Размер холста**: 128x128px
**Биомы**: Кикиморины топи (39-43), Дикое поле (31-38)

### Внешность

Любой базовый враг, окутанный зелёной ядовитой аурой. Кожа приобретает зеленоватый оттенок. Из тела сочится зелёная жидкость — капли стекают по конечностям. Вокруг — постоянное ядовитое облако. Оставляет за собой ядовитый след.

**Ключевой визуальный признак**: зелёное ядовитое облако вокруг тела (radius 20px, `#7FFF00` opacity 30%). Единственный враг с постоянной зелёной аурой.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Ядовитый зелёный | `#2A5A1A` | Кожа, основной тон |
| Ядовитое свечение | `#7FFF00` | Аура, капли, след |
| Тёмная гниль | `#1A3A0A` | Тени, глубина |
| Глаза | `#FF4400` | Глаза (обязательно) |

### run (6 кадров)

- **Кадр 1**: шаг левой, ядовитая аура пульсирует (radius 20px)
- **Кадр 2**: шаг правой, из тела падает капля яда `#7FFF00` (6px)
- **Кадр 3**: промежуточная поза, аура сжимается (radius 16px)
- **Кадр 4**: шаг левой, аура расширяется (radius 24px); ядовитый след за спиной `#7FFF00` opacity 20%
- **Кадр 5**: шаг правой, капля падает
- **Кадр 6**: промежуточная; аура нормализуется

**Длительность цикла**: 0.8 сек (≈0.13 сек/кадр)

### death (4 кадра)

- **Кадр 1**: враг останавливается, аура вспыхивает (radius 32px, opacity 60%)
- **Кадр 2**: тело начинает растворяться снизу; ядовитая лужа `#7FFF00` растекается (48px)
- **Кадр 3**: тело почти исчезло; лужа на максимуме; зелёные частицы (6 штук, 4px) разлетаются
- **Кадр 4**: полное растворение; лужа затухает (opacity 30%); последние частицы

**Длительность**: 0.6 сек (0.15 сек/кадр)

## E_HEALER — Целитель

**ID**: E_HEALER
**Тип**: Модификатор — хил-аура
**Размер холста**: 128x128px
**Биомы**: Пещеры Кощея (44-50), Чёрная крепость (59-65)

### Внешность

Враг со светящимися белыми/золотыми элементами. Вокруг — тёплая хил-аура, лечащая ближайших союзников. На голове — подобие нимба из тёмной энергии (инверсия: тёмное золото вместо светлого). Руки подняты — жест исцеления. Движется медленнее обычного.

**Ключевой визуальный признак**: тёмный нимб `#8B6914` над головой (кольцо 24px diameter, 2px толщина) + золотистые частицы `#FFD700` вокруг (4-5 штук, 3px). Единственный враг с нимбом.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Тёмное золото | `#8B6914` | Нимб, основные акценты |
| Светлое золото | `#FFD700` | Частицы хила, аура |
| Тёмный фон тела | `#3D1A5C` | Тело нечисти |
| Глаза | `#FF4400` | Глаза (обязательно) |

### run (6 кадров)

- **Кадр 1**: шаг левой, нимб стабилен; золотые частицы парят вокруг
- **Кадр 2**: шаг правой, частицы смещаются вниз; хил-аура `#FFD700` opacity 20% radius 24px
- **Кадр 3**: промежуточная, аура пульсирует (radius 28px)
- **Кадр 4**: шаг левой, от целителя летит хил-луч `#FFD700` (1px) к ближайшему союзнику (визуальная линия 48px)
- **Кадр 5**: шаг правой, луч исчезает; частицы перемещаются вверх
- **Кадр 6**: промежуточная; нимб пульсирует (яркость +20%)

**Длительность цикла**: 1.0 сек (≈0.17 сек/кадр — медленнее обычных)

### death (4 кадра)

- **Кадр 1**: нимб вспыхивает `#FFD700` (яркий), тело дрожит
- **Кадр 2**: нимб разлетается на 6 осколков `#8B6914` (8px каждый); тело рассыпается
- **Кадр 3**: осколки разлетаются дальше; тело — пыль `#3D1A5C`
- **Кадр 4**: всё исчезает; последняя вспышка `#FFD700` opacity 30%

**Длительность**: 0.5 сек (0.125 сек/кадр)

## E_GHOST — Призрак

**ID**: E_GHOST
**Тип**: Модификатор — полупрозрачность + мерцание
**Размер холста**: 128x128px
**Биомы**: Полночный предел (51-58), Кикиморины топи (39-43)

### Внешность

Полупрозрачный враг — призрачная версия любого базового типа. Opacity в idle = 50%. Периодическое мерцание — opacity колеблется от 30% до 70%. Контуры размыты. Цвет тела — сине-белый, холодный. Не касается земли — парит в 4-6 пикселях над поверхностью. Оставляет призрачный хвост (ghost trail).

**Ключевой визуальный признак**: полупрозрачность 50% + мерцание. Единственный враг, через которого видно фон. Парит над землёй.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Призрачный голубой | `#A0C4E8` | Основной цвет тела (opacity 50%) |
| Белое свечение | `#E8F0F8` | Блики, контуры |
| Тёмная тень | `#1A2B40` | Тени (opacity 30%) |
| Глаза | `#FF4400` | Глаза — единственное яркое (обязательно, opacity 100%) |

### run (6 кадров)

- **Кадр 1**: парит, opacity 50%; призрачный хвост `#A0C4E8` opacity 20% тянется назад (32px)
- **Кадр 2**: мерцание — opacity 70% (становится более видимым)
- **Кадр 3**: opacity 40%; хвост длиннее (48px); тело смещается на 4px вверх
- **Кадр 4**: opacity 30% (почти невидим); тело на 6px выше обычного
- **Кадр 5**: мерцание обратно — opacity 60%; тело опускается
- **Кадр 6**: opacity 50%; хвост нормализуется; петля

**Длительность цикла**: 1.0 сек (≈0.17 сек/кадр)

### death (4 кадра)

- **Кадр 1**: opacity резко до 100% — призрак становится полностью видимым; выражение ужаса
- **Кадр 2**: тело начинает рассеиваться от центра — 8 призрачных осколков `#A0C4E8` (6px) разлетаются
- **Кадр 3**: осколки тают; центр тела — пустота; контур ещё виден opacity 30%
- **Кадр 4**: полное исчезновение; последняя вспышка `#E8F0F8` opacity 20%

**Длительность**: 0.5 сек (0.125 сек/кадр)

## E_KIKIMORA — Кикимора

**ID**: E_KIKIMORA
**Тип**: Уникальный враг — болотная нечисть
**Акт**: 2 (Кикиморины топи, уровни 39-43)
**Размер холста**: 128x128px
**Размер спрайта**: ~90% холста

### Внешность

Кикимора — болотный дух в виде сгорбленной старухи с длинными спутанными волосами-водорослями. Кожа зеленовато-серая, покрыта тиной и ряской. Длинные тонкие руки с перепонками между пальцами (как у лягушки). Одета в ветхое тёмно-зелёное рубище, промокшее и облепленное болотной тиной. Горб на спине — из него торчат грибы и мох. Движется прыжками, как жаба — приседает и выпрыгивает. Глаза огромные, жёлто-зелёные (не стандартный `#FF4400` — исключение для болотной нечисти), но в темноте светятся `#FF4400`.

**Ключевой визуальный признак**: горб с грибами + перепончатые пальцы + движение прыжками. Единственный враг с горбом.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Болотная кожа | `#4A6B4A` | Лицо, руки, ноги |
| Тина/водоросли | `#1A3B1A` | Волосы, тина на одежде |
| Мокрое рубище | `#2A4A2A` | Одежда |
| Глаза (свечение) | `#FF4400` | Глаза в боевом режиме |

### run (6 кадров)

- **Кадр 1**: приседание — ноги согнуты, руки перед собой; перепонки видны
- **Кадр 2**: начало прыжка — тело выпрямляется, ноги отталкиваются
- **Кадр 3**: в воздухе — тело поджато, рубище развевается; мелкие капли воды `#4A6B4A` opacity 40% (4 штуки)
- **Кадр 4**: приземление — ноги касаются земли, всплеск грязи `#1A3B1A` (8px)
- **Кадр 5**: стабилизация — руки расставлены для баланса; грибы на горбу покачиваются
- **Кадр 6**: подготовка к следующему прыжку — присед; петля

**Длительность цикла**: 0.8 сек (≈0.13 сек/кадр)

### death (4 кадра)

- **Кадр 1**: Кикимора запрокидывает голову назад; глаза гаснут
- **Кадр 2**: тело оседает — превращается в кучу болотной тины `#1A3B1A`; грибы отваливаются
- **Кадр 3**: тина расплывается; из неё поднимаются пузыри `#4A6B4A` opacity 40% (6 штук, 4px)
- **Кадр 4**: только лужа тины осталась `#1A3B1A` opacity 30%; пузыри лопаются

**Длительность**: 0.6 сек (0.15 сек/кадр)

## E_UPYR — Упырь

**ID**: E_UPYR
**Тип**: Уникальный враг — тёмная нечисть
**Акт**: 2 (Пещеры Кощея, уровни 44-50)
**Размер холста**: 128x128px
**Размер спрайта**: ~95% холста

### Внешность

Упырь — мертвец, поднятый тёмной магией Кощея. В отличие от мертвяков Фазы 1 — этот быстрый и хитрый. Тощее тело, длинные когтистые руки. Кожа мертвенно-бледная с синюшным оттенком. Разорванная боярская одежда (когда-то был знатным) — тёмно-пурпурный кафтан в клочьях. Рот раскрыт — видны клыки. Движется на четвереньках — звериная, хищная манера.

**Ключевой визуальный признак**: бег на четвереньках + клыки + пурпурный рваный кафтан. Единственный враг на четвереньках.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Мертвенная кожа | `#6B6B8A` | Тело, лицо |
| Рваный кафтан | `#3D1A5C` | Одежда |
| Тёмная кровь | `#4A1A1A` | Рот, когти, пятна |
| Глаза | `#FF4400` | Глаза (обязательно) |

### run (6 кадров)

- **Кадр 1**: бег на четвереньках — правая рука + левая нога вперёд; тело низко над землёй
- **Кадр 2**: отталкивание — тело приподнимается; рваный кафтан развевается
- **Кадр 3**: промежуточная — все конечности под телом (прыжок)
- **Кадр 4**: левая рука + правая нога вперёд; когти (6px) скребут землю
- **Кадр 5**: отталкивание; рот раскрыт — клыки видны `#F5F0E8` (2 клыка, 4px)
- **Кадр 6**: промежуточная; рот закрывается; петля

**Длительность цикла**: 0.6 сек (0.1 сек/кадр — быстрый)

### death (4 кадра)

- **Кадр 1**: Упырь поднимается на задние лапы — последний рык; рот максимально открыт
- **Кадр 2**: тело трескается — фиолетовые линии `#6B2FA0` по всему телу (трещины тьмы)
- **Кадр 3**: тело рассыпается на тёмные осколки `#3D1A5C` (10 штук, 4-8px)
- **Кадр 4**: осколки растворяются; фиолетовая пыль `#3D1A5C` opacity 20% оседает

**Длительность**: 0.5 сек (0.125 сек/кадр)

## E_ICE_GOLEM — Ледяной голем

**ID**: E_ICE_GOLEM
**Тип**: Уникальный враг — элитный (много HP, медленный)
**Акт**: 3 (Полночный предел, уровни 51-58)
**Размер холста**: 128x128px (тело занимает 95% — массивный)

### Внешность

Огромная гуманоидная фигура, собранная из ледяных глыб. Тело состоит из крупных кусков льда — прозрачных, с синим отливом. В щелях между глыбами — голубое свечение (магическое ядро). Нет лица — вместо головы ледяной валун с двумя горящими красными глазами. Руки массивные, заканчиваются ледяными кулаками. Ноги — колонны льда. При каждом шаге — трещит и скрипит (визуально: мелкие осколки падают с тела).

**Ключевой визуальный признак**: составное тело из ледяных глыб + голубое свечение в щелях. Самый крупный враг (заполняет 95% холста). Единственный враг из льда.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Лёд (основной) | `#A0C4E8` | Глыбы тела |
| Глубокий лёд | `#4A7AA0` | Тени, глубина между глыбами |
| Ядро (свечение) | `#80D4FF` | Свечение в щелях |
| Глаза | `#FF4400` | Глаза (обязательно) |

### run (6 кадров)

- **Кадр 1**: тяжёлый шаг правой ногой — осколки льда `#A0C4E8` (4 штуки, 3px) падают с плеча
- **Кадр 2**: переваливается на левую; тело наклоняется (5 градусов); свечение в щелях пульсирует
- **Кадр 3**: шаг левой — удар ногой о землю; трещины на поверхности `#80D4FF` (2 линии, 12px)
- **Кадр 4**: выпрямление; осколки с другого плеча (3 штуки)
- **Кадр 5**: промежуточная — оба кулака сжаты, тело прямо; свечение ослабевает
- **Кадр 6**: подготовка к шагу; свечение усиливается; петля

**Длительность цикла**: 1.2 сек (0.2 сек/кадр — очень медленный, тяжёлый)

### death (4 кадра)

- **Кадр 1**: трещина проходит через всё тело от головы до ног — `#80D4FF` (яркая линия 2px)
- **Кадр 2**: тело разваливается на 4 крупных куска `#A0C4E8` (32x24px каждый); свечение вырывается
- **Кадр 3**: куски разлетаются и тают; вспышка `#80D4FF` radius 24px
- **Кадр 4**: только мелкие осколки и лужица `#A0C4E8` opacity 30%; свечение гаснет

**Длительность**: 0.6 сек (0.15 сек/кадр)

## E_DARK_KNIGHT — Тёмный витязь

**ID**: E_DARK_KNIGHT
**Тип**: Уникальный враг — элитный (бронированный, средняя скорость)
**Акт**: 3 (Чёрная крепость, уровни 59-65)
**Размер холста**: 128x128px
**Размер спрайта**: ~95% холста

### Внешность

Рыцарь тьмы — бывший защитник Руси, павший и порабощённый Кощеем. Чёрные латы из обсидиана с красными рунами. Шлем с рогами (2 загнутых назад рога). Двуручный чёрный меч, лезвие светится красным. Красный плащ, рваный и обожжённый. Двигается уверенно — воинская выправка сохранилась. Глаза за забралом горят `#FF4400` — яркие и осмысленные (он знает, что делает).

**Ключевой визуальный признак**: рогатый шлем + чёрные латы + красный светящийся меч. Единственный враг с рогатым шлемом и мечом.

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Обсидиановый чёрный | `#1A0A1A` | Латы |
| Кровавый красный | `#C00020` | Руны, свечение меча, плащ |
| Тёмный металл | `#0D0D0D` | Тени лат |
| Глаза | `#FF4400` | Глаза за забралом |

### run (6 кадров)

- **Кадр 1**: маршевый шаг — правая нога вперёд; меч на плече; плащ развевается
- **Кадр 2**: промежуточная — тело выпрямлено; руны на латах пульсируют `#C00020`
- **Кадр 3**: шаг левой; меч чуть опустился; лезвие светится ярче
- **Кадр 4**: промежуточная — тело наклонено вперёд (5 градусов); уверенная поступь
- **Кадр 5**: правая нога; плащ взметнулся; руны затихают
- **Кадр 6**: выпрямление; петля

**Длительность цикла**: 0.8 сек (≈0.13 сек/кадр)

### death (4 кадра)

- **Кадр 1**: Тёмный витязь останавливается; красное свечение меча гаснет; глаза тускнеют
- **Кадр 2**: латы трескаются — красные трещины `#C00020` по всему телу; меч падает
- **Кадр 3**: латы рассыпаются; внутри — пустота (призрачный силуэт `#3D1A5C` opacity 40%); красные частицы (8 штук, 4px) разлетаются
- **Кадр 4**: всё исчезает; на земле — куча чёрных осколков `#1A0A1A` и затухающие руны

**Длительность**: 0.6 сек (0.15 сек/кадр)

## E_FIRE_ELEM — Огненный элементаль

**ID**: E_FIRE_ELEM
**Тип**: Уникальный враг — элитный (DoT урон, средний HP)
**Акт**: 3 (Чёрная крепость 59-65, Пещеры Кощея 44-50)
**Размер холста**: 128x128px
**Размер спрайта**: ~85% холста (огненное тело без чётких границ)

### Внешность

Существо из чистого огня — гуманоидная форма, но без деталей. Ядро — яркий жёлто-белый центр в груди. Тело — вихрь пламени: внутренние слои ярко-жёлтые, внешние — оранжевые и красные. Нет лица — только два ярких глаза `#FF4400` в «голове» из огня. Конечности — языки пламени, нечёткие. Парит над землёй — под ним выжженный круг. Постоянно меняет форму (каждый кадр — разные контуры огня).

**Ключевой визуальный признак**: тело из чистого огня + яркое ядро в груди. Единственный враг без чётких контуров (огненное размытие по краям).

### Цветовая схема

| Название | HEX | Применение |
|---------|-----|-----------|
| Ядро (горячий центр) | `#FFD700` | Ядро в груди, самый яркий |
| Внутреннее пламя | `#FF6D00` | Тело ближе к ядру |
| Внешнее пламя | `#C62828` | Внешние языки огня |
| Глаза | `#FF4400` | Глаза (обязательно) |

### run (6 кадров)

- **Кадр 1**: парит, ядро `#FFD700` radius 12px; языки пламени `#FF6D00` направлены вверх-назад
- **Кадр 2**: сдвиг вперёд; ядро пульсирует (radius 14px); форма тела меняется — левая рука-пламя длиннее
- **Кадр 3**: сдвиг; форма снова другая — правая рука длиннее; внешний слой `#C62828` разлетается (4 искры, 3px)
- **Кадр 4**: ядро ярче (radius 16px); под ним — выжженный след `#3A2510` opacity 40% (16px)
- **Кадр 5**: форма сжимается — более компактный; ядро radius 10px; огонь прижат к телу
- **Кадр 6**: форма расширяется; ядро нормализуется; петля

**Длительность цикла**: 0.6 сек (0.1 сек/кадр — быстрая анимация огня)

### death (4 кадра)

- **Кадр 1**: ядро вспыхивает `#FFFFFF` (ослепительно); тело разрастается в 1.5 раза
- **Кадр 2**: взрыв — ядро лопается; огненные частицы `#FF6D00` + `#FFD700` (12 штук, 4-8px) разлетаются во все стороны
- **Кадр 3**: частицы рассеиваются; на месте врага — дым `#333333` opacity 50%
- **Кадр 4**: дым рассеивается; остаётся выжженное пятно `#3A2510` opacity 20%

**Длительность**: 0.5 сек (0.125 сек/кадр)
