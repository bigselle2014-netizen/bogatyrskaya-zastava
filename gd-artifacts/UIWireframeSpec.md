# UI Wireframe Spec — Богатырская Застава
**Проект**: Богатырская Застава  
**Дата**: Апрель 2026  
**Статус**: Фаза 1, Недели 2–3  
**Передаётся**: Unity Dev Middle вместе с CoreMechanicsDocument.md

Все размеры даны для вертикального экрана 1080×1920 (соотношение 9:16).  
Отступ от краёв экрана — Safe Area (учитываем notch/punch-hole).

---

## Экран 1: HUD боя (во время волны)

```
┌─────────────────────────────────────┐
│  [HP ворот ████████░░] 850/1000     │  ← Верх-лево, высота 8% экрана
│                         Волна 3/7   │  ← Верх-право
│                                     │
│                                     │
│        [ИГРОВОЕ ПОЛЕ]               │
│        (центр экрана, 75% высоты)   │
│                                     │
│  [S-01 🔥] [S-07 ⚡]               │  ← Активные синергии, верх-центр под HP
│                                     │
│ ┌────────────────────────────────┐  │
│ │ [DR1] [VL1] [LU1] [BE1] [ZN1] [+]│  ← Дека, низ экрана, высота 15%
│ │  50₽   55₽   55₽   50₽   60₽  │  │
│ └────────────────────────────────┘  │
└─────────────────────────────────────┘
```

### Детальное описание элементов HUD:

**HP ворот**
- Позиция: верх-лево, отступ 16px от Safe Area
- Размер: 45% ширины экрана × 8% высоты
- Элементы: прогресс-бар (красный → зелёный), текст "850 / 1000"
- Анимация: тряска при получении урона

**Счётчик волны**
- Позиция: верх-право, симметрично HP
- Формат: "Волна 3 / 7"
- Размер шрифта: 24sp

**Таймер между волнами** (виден только в паузе между волнами)
- Позиция: верх-центр, под строкой HP/волны
- Формат: обратный отсчёт "13" → "0" (10–15 секунд)
- Размер: 48sp жирный, цвет золотой
- Исчезает при старте волны

**Иконки активных синергий**
- Позиция: под таймером, центр, горизонтальная полоска
- Максимум 3 иконки одновременно (по game-bible — 12 синергий, но одновременно активных мало)
- Размер иконки: 48×48dp
- При наведении (долгий тап 0.5с): всплывает описание синергии

**Дека башен (нижняя панель)**
- Позиция: низ экрана, полная ширина, высота 15% (~288px)
- 6 слотов (по game-bible: максимум 6 башен в деке)
- Каждый слот: иконка башни 64×64dp + стоимость под ней
- Слот недоступен (нет монет): затемнение 50% + иконка монеты с крестом
- Слот уже размещён: пульсирующая зелёная рамка
- 7-й слот "+": кнопка смены башни (если открыты >6 башен)

## Экран 2: Расстановка башен (до старта волны)

```
┌─────────────────────────────────────┐
│  Монеты: 320 🪙     Камни: 5 💎    │  ← Верх, ресурсы
│                                     │
│  ┌─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┬─┐       │
│  │ │ │ │ │ │ │ │ │ │ │ │ │ │       │
│  ├─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┼─┤       │
│  │ │▓│ │ │▓│ │ │ │▓│ │ │ │ │  ← Сетка поля 8×12
│  ├─┼─...                   │       │  (▓ = занято башней)
│  ...                        │       │
│  └─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┴─┘       │
│                                     │
│  [Синергия активна: S-01 +30% урон] │  ← Баннер синергии (зелёный)
│                                     │
│  ┌────────────────────────────────┐ │
│  │[DR1][VL1][LU1][__][__][__]    │ │  ← Дека
│  └────────────────────────────────┘ │
│        [ ▶ НАЧАТЬ ВОЛНУ ]           │  ← Кнопка, низ-центр
└─────────────────────────────────────┘
```

### Детальное описание:

**Сетка поля**
- Размер: 8 колонок × 12 строк (вертикальная ориентация, соотношение 2:3)
- Занимает 70% высоты экрана, 95% ширины
- Занятые башней клетки: иконка башни в клетке
- Свободные клетки: лёгкая сетка (opacity 20%)
- Путь врагов: визуально выделен (тёмная полоса / трава другого цвета — не кликабельные клетки)
- При выборе башни из деки: доступные клетки подсвечиваются зелёным, путь врагов — серым

**Индикатор синергии**
- Появляется когда размещены башни, образующие синергию
- Баннер под сеткой: "Синергия: [название] — [краткий эффект]"
- Подсветка: зелёная рамка вокруг башен, участвующих в синергии
- При размещении башни-соседа, несовместимой с соседом: никакой подсветки (просто нет синергии)

**Кнопка "Начать волну"**
- Позиция: самый низ, центр, над декой
- Размер: 70% ширины × 56dp высота
- Цвет: золотой / активный
- Disabled если дека пустая (нет ни одной башни на поле)

**Ресурсы (верхняя строка)**
- Монеты + Рунные камни, иконки + цифры
- Обновляются в реальном времени при покупке/продаже башен

## Экран 3: Конец уровня (победа)

```
┌─────────────────────────────────────┐
│                                     │
│           ✦ ✦ ✦                    │  ← 3 звезды (анимация появления)
│        ПОБЕДА!                      │
│                                     │
│  ┌─────────────────────────────┐    │
│  │ + 200 монет 🪙               │    │
│  │ + 3 рунных камня 💎          │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐    │  ← Кнопка rewarded video
│  │  📺 × 2 ресурсы  (11 сек)  │    │    Исчезает через 3 сек
│  └─────────────────────────────┘    │
│                                     │
│  [← В меню]    [Следующий уровень →]│  ← Кнопки
└─────────────────────────────────────┘
```

### Детальное описание:

**Звёзды**
- 3 звезды: HP ворот ≥ 80% от максимума
- 2 звезды: HP ворот 40–79%
- 1 звезда: HP ворот 1–39%
- Анимация: звёзды падают сверху по очереди с задержкой 0.4с

**Блок наград**
- Монеты и рунные камни с иконками
- Анимация: счётчик накручивается от 0 до финального значения (0.5с)

**Кнопка Rewarded Video**
- Видна 3 секунды (таймер обратного отсчёта на кнопке: "11 сек", "10 сек"...)
- Исчезает по истечении таймера (не остаётся серой — именно исчезает)
- По game-bible: не чаще 1 раза в 10–15 минут — если лимит исчерпан, кнопка не показывается
- Эффект: ×2 к полученным наградам этого уровня

**Кнопки навигации**
- "В меню": левая, вторичная (меньше, без акцента)
- "Следующий уровень": правая, первичная (золотой, большой)
- Если следующий уровень не открыт: кнопка "Следующий" disabled, показывает "Уровень закрыт"

## Технические заметки для Unity Dev Middle

- Все экраны реализуются как отдельные Canvas / Panel в UIManager
- Переходы между экранами: через GameStateMachine (реализован Lead Dev'ом)
- Шрифт: кириллический, поддерживающий жирный и курсив
- Safe Area: обязательно использовать `Screen.safeArea` для отступов
- Анимации: простые DOTween / встроенный Animator (без сложных rig-систем)
- Масштабирование: Canvas Scaler → Scale With Screen Size, Reference 1080×1920
