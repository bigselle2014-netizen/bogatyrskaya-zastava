# TowerDesignSheet — Богатырская Застава
**Версия:** 1.0 | **Дата:** Апрель 2026
**Хендовер:** 2D Художник — Неделя 2

---

## Технические требования к спрайтам

- Размер холста: **256×256 пикселей**
- Формат: **PNG → ASTC 6×6** (сжатие под Android)
- Состояний на каждую башню: **3 (idle / attack / upgrade)**
- Baseline устройство: Redmi A2+, дисплей ~720p
- Стиль: **славянское фэнтези, 2D, вид сбоку/3/4 сверху, лёгкая стилизация (не гиперреализм)**

---

## Описание состояний

| Состояние | Описание |
|-----------|---------|
| **idle** | Башня стоит, спокойно дышит / покачивается (1–2 кадра анимации петли) |
| **attack** | Момент атаки: замах, выстрел, вспышка способности (3–5 кадров) |
| **upgrade** | Вспышка энергии / обновление облика: добавляется деталь брони, блеск (2–3 кадра) |
