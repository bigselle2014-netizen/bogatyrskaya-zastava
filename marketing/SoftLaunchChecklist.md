## Блок 1: Юридическое

- [ ] **ООО зарегистрировано** — без юрлица RuStore не подключает платёжную систему. Минимальный уставный капитал 10 000₽. Открытие через Госуслуги ~3–5 рабочих дней.
- [ ] RuStore Publisher Agreement подписан — срок обработки ~2 недели, подавать не позднее чем за 2 недели до плановой даты публикации.
- [ ] **IT-аккредитация получена** — необходима для льготного налогообложения (налог на прибыль 0% для IT, страховые взносы 7.6%). Подача через Минцифры. Срок рассмотрения ~30 дней.
- [ ] **Товарный знак «Богатырская Застава» подан на регистрацию** — заявка в Роспатент. Регистрация занимает 18–24 месяца, но дата приоритета фиксируется при подаче. Стоимость ~33 000₽ для 1 класса МКТУ.
- [ ] **Договоры с командой подписаны** — все участники (штат и фриланс) подписали договоры об отчуждении исключительных прав. IP принадлежит ООО, а не физическим лицам.
- [ ] **Политика конфиденциальности опубликована** — обязательное требование RuStore. URL должен быть доступен публично (можно на сайте или GitHub Pages).
- [ ] **Пользовательское соглашение опубликовано** — отдельный документ. Должен включать описание монетизации, политику возврата, возрастные ограничения.

**Ответственный**: CEO / Founder
**Дедлайн**: месяц 3 (параллельно с разработкой)

## Блок 3: Технический

### Аналитика
- [ ] **AppMetrica SDK установлен и верифицирован** — версия актуальная. API Key в BuildSettings.
- [ ] **Воронка событий настроена в AppMetrica**:
  - `install` → `first_open` → `tutorial_complete` → `level_1_complete` → `first_rewarded_ad_shown` → `first_rewarded_ad_watched` → `first_iap_initiated` → `first_iap_complete`
  - `tutorial_start` — начало туториала
  - `tutorial_complete` — завершение туториала (нужно для гейта Фазы 1: D1 self-report ≥70%)
  - `level_complete {levelNumber, stars, timeSeconds}` — прохождение уровня
  - `synergy_activated {synergyId}` — активация синергии (ключевая механика)
  - Проверить что каждое событие отправляется корректно (QA верифицирует на тестовом устройстве)
- [ ] **D1/D7 Retention настроены как когорты в AppMetrica**
- [ ] **myTracker SDK установлен** — для VK Ads атрибуции. Токен в BuildSettings. Проверить: install-событие корректно передаётся в VK Ads.
- [ ] **Дашборд AppMetrica настроен** — воронка install→first_iap видна на главном экране. Алерты при CR(tutorial_complete) < 70%.

### Монетизация
- [ ] **RuStore Pay SDK v10.1.1 интегрирован** — версия строго 10.1.1 (согласно game-bible). Sandbox-тест пройден: тестовая покупка проходит, refund проходит.
- [ ] **IAP-продукты созданы в RuStore консоли** — минимальный набор: стартовый пак (149₽), Battle Pass месяц (249₽), ускорение ×2 idle (99₽). Цены проверены с менеджером RuStore.
- [ ] **Rewarded Video настроен** — SDK рекламной сети подключён. Тест: rewarded video загружается и воспроизводится на Xiaomi Redmi A2+. Таймаут загрузки < 3 сек.
- [ ] **Fallback при отсутствии рекламы** — если нет заполненности, кнопка «Смотреть рекламу» не показывается (не серая кнопка которую нельзя нажать).

### Качество
- [ ] **Crash rate < 1% на Xiaomi Redmi A2+** — базовое устройство тестирования. Прогнать 50 сессий по core loop, замерить крэши через Firebase Crashlytics или AppMetrica.
- [ ] **Размер APK ≤ 100MB** — для пользователей без Wi-Fi. Использовать ASTC 6x6 (согласно game-bible).
- [ ] **Загрузка < 3 сек на первом экране** — замерить на Xiaomi Redmi A2+ с 4G (без Wi-Fi).
- [ ] **FPS ≥ 30 во время волн** — замерить peak-load на 3 волне Берёзового тракта с 6 башнями и синергиями.
- [ ] **Офлайн-режим работает корректно** — игра запускается без интернета. Idle-накопление считается корректно после 8+ часов офлайна.

**Ответственный**: Lead Dev + QA
**Дедлайн**: месяц 4 (за 4 недели до Soft Launch)

## Блок 5: Метрики — gate до запуска платного UA

> Эти метрики — жёсткий gate. Если хотя бы одна не достигнута — СТОП. Итерация продукта, не запуск UA. Источник: game-bible v1.0.

- [ ] **D1 Retention ≥ 35%** — измерено на выборке ≥ 20 тестеров. Период: последние 3 дня (не пиковые значения). Источник данных: AppMetrica когорты.
- [ ] **D7 Retention ≥ 12%** — измерение доступно только с 7-го дня после старта. Мягкий лонч начинается в месяц 5 нед 1; финальное решение о переходе к Scale принимается в месяц 5 нед 4 (не нед 2).
- [ ] **IAP conversion ≥ 3%** — доля пользователей из когорты, совершивших хотя бы одну IAP-покупку. Источник: AppMetrica + RuStore консоль.
- [ ] **Все три метрики достигнуты одновременно** — недостаточно иметь D1 40% если D7 = 10%. Три порога одновременно — обязательное условие.
- [ ] **Если метрики не достигнуты → зафиксировать причину** — гипотезы: туториал непонятен / синергии не очевидны / idle накопление слишком медленное / монетизация появляется слишком рано.

**Период измерения**: последние 3 дня перед принятием решения о Soft Launch. Не средние за всё время.
**Ответственный за решение**: Product Lead / CEO
**Дедлайн решения**: месяц 5, неделя 4
