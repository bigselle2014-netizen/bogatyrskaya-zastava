## 1. Цель Playable Ad

Игрок за 30 секунд понимает главную механику игры (синергии башен) и хочет установить. Интерактивность создаёт ощущение «я уже играю» — снижает барьер установки.

**Целевое действие**: нажать кнопку «Играть бесплатно» → переход в RuStore.

## 2. Полный сценарий интерактива (30 сек)

### Экран 1 — Загрузочный (0–2 сек)

**Что показывает**:
- Логотип «Богатырская Застава» на тёмном фоне (#1A1A2E)
- Под логотипом: «Попробуй прямо сейчас»
- Прогресс-бар загрузки (анимация 2 секунды)

**Переход**: автоматически → Экран 2

---

### Экран 2 — Поле боя, исходное состояние (2–6 сек)

**Что показывает**:
- Упрощённое поле: 4×4 клетки. Биом «Берёзовый тракт» (фон — зелёный, берёзы, дорожка посередине)
- На поле уже стоят: **Буян (BE-2)** — левая нижняя клетка, **Чародей (VL-2)** — правая нижняя клетка
- Между башнями: пульсирующий вопросительный знак «?»
- Вверху: HP-бар Заставы (100%), счётчик «Волна 1/3»
- Снизу: подсказка «Нечисть идёт! Расставь богатырей»
- Из правого края кадра выходит волна из 6 Кикимор (быстрые враги)

**Взаимодействие**: нет. Игрок наблюдает — напряжение нарастает.

---

### Экран 3 — Синергия S-07 (6–12 сек)

**Что показывает**:
- Кикиморы приближаются к Буяну и Чародею
- Автоматически вспыхивает эффект синергии: линия света соединяет Буяна и Чародея
- Крупно появляется иконка **«S-07 Огненный шар»** в центре экрана на 1.5 сек с текстом «Синергия активирована!»
- Буян выпускает огненный шар — AOE взрыв по всем Кикиморам
- Все враги уничтожены. Монеты сыплются.
- Текст внизу: «Собери богатырей рядом — и они усилят друг друга!»

**Взаимодействие**: нет. Игрок видит результат автоматической синергии.

**Звук (опционально)**: звук взрыва, звон монет. Должно работать БЕЗ звука (субтитры/текст передают смысл).

---

### Экран 4 — Действие игрока: поставить башню (12–22 сек)

**Что показывает**:
- Волна 2: 8 Кикимор + 1 Лесной тролль (медленный, но крепкий)
- На поле появляется пустая клетка с золотой подсветкой в левом верхнем углу
- Пульсирующая стрелка указывает на клетку
- В нижней части экрана — иконка башни **Травник (ZN-1)** с подписью «Поставь Травника»
- Текст: «Тапни на клетку» с анимацией пальца

**Взаимодействие — ЕДИНСТВЕННОЕ ДЕЙСТВИЕ ИГРОКА**:
- Игрок тапает по подсвеченной клетке
- Травник появляется на поле
- Автоматически активируется **S-08 Ядовитое исцеление** (Соколиный глаз рядом активируется автоматически — он уже был на поле)
- Текст: «S-08: Яд лечит башни! Соколиный глаз + Травник»
- Враги уничтожены. Счётчик: «Волна 2/3 ✓»

**Если игрок не тапает 3 секунды**: автоматическая демонстрация (auto-tap анимация) — не блокируем прогресс.

---

### Экран 5 — Финальная волна (22–27 сек)

**Что показывает**:
- Волна 3 (финальная): толпа врагов + мини-босс в конце
- Все 4 башни на поле: Буян (BE-2), Чародей (VL-2), Травник (ZN-1), Соколиный глаз (LU-2)
- S-07 и S-08 срабатывают одновременно. Вся толпа уничтожена.
- Мини-босс замедляется (S-09 Ледяной щит — визуальный эффект заморозки)
- Анимация: «Застава выстояла!» золотыми буквами

**Взаимодействие**: нет.

---

### Экран 6 — CTA (27–30 сек + удержание)

**Что показывает**:
- Тёмный фон (#1A1A2E)
- Логотип «Богатырская Застава» — крупно, с золотым свечением
- Текст: «15 богатырей. 12 синергий. Бесплатно.»
- Подтекст: «Без доната · Без gacha · Честная игра»
- Кнопка — зелёная, крупная: **«Играть бесплатно»**
- Под кнопкой: иконка RuStore + «Скачать в RuStore»

**Взаимодействие**: тап по кнопке → deep link в RuStore на страницу игры.

**Удержание**: экран остаётся, не закрывается автоматически. Ждём действия.

## 3. Технические ограничения VK Ads Playable

| Параметр | Требование |
|----------|-----------|
| Размер файла | До 2MB (HTML5 + все assets включены) |
| Формат | HTML5 (один .zip архив) |
| Разрешения | 9:16 (1080×1920px) — основной; 16:9 (1920×1080px) — для десктоп |
| Framerate | Минимум 30 FPS на Xiaomi Redmi A2+ (тестовое устройство) |
| Звук | Опционален. Playable должен быть полностью понятен БЕЗ звука |
| Интерактивность | Максимум 3 действия пользователя. В нашем сценарии: 1 действие (tap-to-place) |
| Сетевые запросы | ЗАПРЕЩЕНЫ. Все ресурсы — в архиве. Нет обращений к серверу. |
| JavaScript | Разрешён. Анимации — CSS + Canvas (не WebGL для совместимости) |
| Шрифты | Embed в архив. Google Fonts не загружать. |
| CTA кнопка | Должна быть видна в последние 5 секунд. Deep link на RuStore. |
| Android совместимость | Chrome 90+, Android WebView 90+ |
